## Usage


Pretty easy!

Open up the included [Xcode project](GMDKit.xcodeproj) for an [example app](Example).

## Installation
### CocoaPods

Add the following to your `Podfile`:

``` ruby
pod 'GMKit', :git => 'git@github.com:gabemdev/GMDKit.git'
```

Then run `pod install`.
